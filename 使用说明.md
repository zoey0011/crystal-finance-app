# 水晶店财务助手使用说明

这是一个适合手机和平板使用的水晶店财务小程序，包含商品、进货、销售、开支、库存和月度利润统计。

## 怎么使用

1. 用浏览器打开 `index.html`。
2. 先在“商品”里维护商品编号、名称、规格和默认进货价。
3. 每次进货，在“进货”里填写日期、商品、数量、进货单价和运费。
4. 每次销售，在“销售”里填写日期、商品、数量、销售单价和订单号。
5. 日常费用在“开支”里记录，比如租金、包装、物流、平台佣金、人工等。
6. “总览”会自动计算当月销售额、销售成本、开支、净利润和库存。

## 导入航富通销售表

1. 在航富通里导出“销售订单明细报表”，文件格式可以是 `.xls` 或 `.xlsx`。
2. 打开本程序的“销售”页面。
3. 点击“选择销售表导入”，选择航富通导出的销售表。
4. 程序只读取里面准确的“实际销售额”，并按销售日期汇总为“航富通销售收入”。
5. 已经导入过的同一订单同一商品会自动跳过，避免销售额重复计算。

导入后请到“总览”查看销售额变化。航富通导入记录不会扣库存，也不会计算商品成本；如果要算准确利润，需要另外在“进货”和“开支”里维护成本与费用。

## 手机和平板同步

当前程序已经支持 Supabase 自动同步。要实现手机和平板同步，需要先创建一个 Supabase 项目，并建立数据表。

在 Supabase 的 SQL Editor 执行：

```sql
create table if not exists crystal_finance_data (
  store_code text primary key,
  data jsonb not null,
  updated_at timestamptz not null default now()
);

alter table crystal_finance_data enable row level security;

create policy "allow anon read"
on crystal_finance_data
for select
to anon
using (true);

create policy "allow anon insert"
on crystal_finance_data
for insert
to anon
with check (true);

create policy "allow anon update"
on crystal_finance_data
for update
to anon
using (true)
with check (true);
```

然后在程序“设置”里填写：

- `Supabase URL`
- `Anon Key`
- `店铺同步码`

手机和平板填写同一组信息后，点“同步”即可共用同一份数据。

配置完成后，程序会自动同步：

- 打开程序时自动拉取最新数据
- 从其他 App 回到程序时自动拉取最新数据
- 每次保存商品、进货、销售、开支后自动上传
- 导入航富通销售额后自动上传
- 页面保持打开时约每 1 分钟自动检查一次

右上角“同步”按钮仍然保留，可以在网络不好或想马上同步时手动点一下。

## 添加到手机桌面

### iPhone / iPad

1. 用 Safari 打开程序网址。
2. 点分享按钮。
3. 选择“添加到主屏幕”。

### Android

1. 用 Chrome 打开程序网址。
2. 点右上角菜单。
3. 选择“添加到主屏幕”或“安装应用”。

## 备份

在“设置”里可以导出 JSON 备份文件，也可以把备份文件重新导入。

建议每周导出一次备份，避免误删或换设备时丢失数据。
